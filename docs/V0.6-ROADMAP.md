| Vergleich | Realität |

## 2. v0.6.0 — Was jetzt drin ist

### ✅ Multi-Agent Conductor (Standard)

`CONDUCTOR_ORCHESTRATION=multi` (Default) — drei parallele LLM-Calls:

```
[Prompt] ─┬─► Audio Agent  → Strudel + music-constraints + Acorn
          ├─► Video Agent  → Hydra
          └─► Synth Agent  → WAM JSON
                    ↓
              gleiches Manifest wie v0.5
```

`CONDUCTOR_ORCHESTRATION=mono` — alter Einzel-JSON-Call.

Dateien: `server/conductor-agents.mjs`, `server/llm-call.mjs`, `server/conductor.mjs`

### ✅ Acorn Syntax-Guard

`server/code-validate.mjs` — vor REPL:

- `/api/generate` — Fallback auf `previousCode` bei Syntaxfehler
- `/api/conduct`, `/api/ignite` — 422 oder Fallback wenn `fallbackStrudel` mitgegeben
