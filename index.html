Strudel Live

Beschreibe den Sound in normaler Sprache — die KI schreibt Strudel-Code und spielt ihn ab. Oder wähle ein Starter-Pattern und feile per Hand nach (Ctrl+Enter).

Einmaliges Setup: cp .env.example .env — dann OPENAI_API_KEY oder ANTHROPIC_API_KEY eintragen und npm run dev neu starten.

Erweitert: Mikrofon, MIDI & Samples
  • MIDI → Ableton: Pattern 04-midi-ableton — siehe docs/MIDI-MAC.md
  • Eigene Samples: npm run dev:full — Dateien in samples/
  • SuperDirt/OSC: docs/SUPERCOLLIDER.md
