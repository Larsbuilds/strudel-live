    Strudel Editor

    Prompt links · Code rechts — direkt editierbar

    Editor-Funktionen
    • // Kommentar — Zeilen mit // werden ignoriert
    • stack(a, b, c) — Schichten parallel (Kick, Bass, Hats …)
    • setcpm(32) — Tempo (32 ≈ 128 BPM bei 4/4)
    • s("bd*4") — Drums · note("c2") — Töne
    • .gain() · .lpf() · .room() · .delay() — Effekte
    • Play-Button im Editor oder Ctrl+Enter
    • strudel.cc/learn
    Wie KI mit dem Editor verbunden ist
    1. Semantic — „mehr Bass“ → Intent-DB (Bedeutung, kein Strudel)
    2. Catalog — Intent → Konzept → Strudel-Funktionen (s(), note()…)
    3. Agent-Tools — append_layer führt den Catalog aus; kleine KI orchestriert JSON-Tools
    4. RAG — relevante Strudel-Snippets im Prompt · Debug: /api/catalog, /api/agent/plan
    5. Presets für ganze Songs · KI für kreative Sprache · Verlauf (× rückgängig)
